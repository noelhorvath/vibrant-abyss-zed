## Description

**Vibrant Abyss** is a high-contrast **Zed** theme designed for optimal clarity and readability against a true pitch-black background.
It's based on the original [Vibrant Abyss for **Visual Studio Code**](https://github.com/noelhorvath/vibrant-abyss).

## Features

- Dark theme
- Pitch black
- Vibrant colors
